### About

I build software at the intersection of AI, automation, and high risk payments infrastructure. Day to day, that means designing AI powered backends, integrating payment rails across LATAM, and shipping automation pipelines that replace manual work with reliable systems.

<br>

### What I work on

**AI and LLM systems.**
Building RAG pipelines, multi agent orchestration, and production LLM applications. I work across the model spectrum, from frontier APIs like Claude, GPT, and Gemini to local models running on Ollama such as Llama, Qwen, Gemma, and Mistral. Most of my LLM stack is built on FastAPI, Pydantic, LangChain, LlamaIndex, and ChromaDB.

**Automation and workflow engineering.**
n8n, Make, Apps Script, and custom Python services. I usually start with off the shelf platforms and graduate to code when the complexity stops fitting in a visual editor.

**Payments and iGaming.**
KYC and AML flows, merchant onboarding, transaction engines, fraud scoring, and regulatory work for LATAM (Brazil's SPA/MF framework and Lei 14.790/2023). PIX, cards, crypto rails, and alternative payment methods.
